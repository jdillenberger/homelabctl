Install Instructions
Linux

Run this command on the target machine:

curl -sSL {{.InstallURL}}/ca/install.sh | sudo bash
macOS

Run this command in Terminal:

curl -sSL {{.InstallURL}}/ca/install.sh | sudo bash
Windows

Run this command in PowerShell as Administrator:

Invoke-WebRequest -Uri "{{.InstallURL}}/ca/cert" -OutFile "$env:TEMP\homelabctl-ca.crt"; Import-Certificate -FilePath "$env:TEMP\homelabctl-ca.crt" -CertStoreLocation Cert:\LocalMachine\Root
iOS
  1. Scan the QR code above or tap Download Certificate
  2. Open Settings → General → VPN & Device Management
  3. Tap the downloaded profile and install it
  4. Go to Settings → General → About → Certificate Trust Settings
  5. Enable full trust for homelabctl Local CA
Android
  1. Scan the QR code above or tap Download Certificate
  2. Open Settings → Security → Encryption & credentials
  3. Tap Install a certificate → CA certificate
  4. Select the downloaded file
